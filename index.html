HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Angled brackets are used in opening and closing tags, with closing tags containing a forward slash.
  • Attributes define additional information about the element and are always in the beginning.
  • An element starts at the opening tag and ends in the closing tag.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain a property and the value of a propety.
  • * Selector delcares all elements on the page will have certain style properties
  • External CSS keeps all CSS rules in a separate file, which needs to be linked to HTML in "head".

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin branch-name: receives the latest information from a certain branch.
  • git push origin branch-name: pushes changes to remote branch.
  • git add -A: commits all files in the working branch.
  • git commit -m "detailed message here": will leave a message on what change was made.
  • Login to Github to compare and pull requests.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements such as "if" will only execute if true and they interrupt control flow.
  • In order to execute a function properly, we have to define it, and call it by name.
  • Functions are used to define and reuse blocks of code.
